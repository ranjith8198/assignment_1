# 🚓 SecureCheck: Police Check Post Dashboard

## 📌 Project Description

SecureCheck is a Python–SQL based digital ledger system designed for police check posts to record, monitor, and analyze vehicle stops.

The system stores police stop records in a SQL database and provides a real-time analytics dashboard built using Streamlit.

It helps officers quickly identify suspicious vehicles, monitor violations, and analyze traffic stop patterns.

---

## 🛠 Technologies Used

* Python
* Pandas
* SQLAlchemy
* PostgreSQL
* Streamlit

---

## 🚀 Features

* Vehicle search system
* Traffic stop summary generator
* Violation analytics
* Driver demographic analysis
* Location-based insights
* Time-based traffic stop analysis

## 📊 Dataset

The dataset contains police traffic stop records including:

* Stop date and time
* Driver age and gender
* Driver race
* Violation type
* Search conducted
* Stop outcome
* Arrest status
* Stop duration
* Drug related stop

This data is used to analyze traffic violations and generate insights for check post monitoring.

---

## ▶️ Run the Project

Install dependencies:

pip install -r requirements.txt

Run the dashboard:

streamlit run app.py
