# SecureCheck: Police Check Post Dashboard

## Project Description
SecureCheck is a Python-SQL based digital ledger system designed for police check posts to track vehicle stops and analyze violations.

## Technologies Used
- Python
- Pandas
- SQLAlchemy
- PostgreSQL
- Streamlit

## Features
- Vehicle search system
- Stop summary generator
- Violation analytics
- Demographic analysis
- Location-based insights
- Time-based analysis

## Run the Project

Install dependencies

pip install -r requirements.txt

Run the Streamlit dashboard

streamlit run app.py
